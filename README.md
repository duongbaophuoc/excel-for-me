# 📊 Excel & Google Sheets AI Playbook

> **Bilingual Reference | Tài liệu Song ngữ Anh–Việt**  
> Inspired by *EXCEL Formulas Bible* — extended with AI integration, real-world case studies, and Google Sheets compatibility.

## 🇬🇧 English

A comprehensive, open-source playbook for mastering Excel and Google Sheets — from beginner formulas to advanced financial modeling and AI-powered automation.

### What's Inside

| Section | Description |
|---|---|
| [`docs/fundamentals/`](docs/fundamentals/) | Core formulas, shortcuts, and UI navigation |
| [`docs/advanced/`](docs/advanced/) | Multi-sheet workflows, performance, VBA, dashboards |
| [`docs/domains/`](docs/domains/) | Finance, accounting, data analysis, forecasting, data science |
| [`docs/ai-integration/`](docs/ai-integration/) | ChatGPT, Gemini, Claude inside spreadsheets |
| [`dashboards/`](dashboards/) | Ready-to-use dashboard templates |
| [`case-studies/`](case-studies/) | Real-world applied examples |

### Quick Start

```
1. Browse docs/fundamentals/formulas/ for formula references
2. Pick a domain in docs/domains/ that matches your use case
3. Check docs/ai-integration/ to supercharge your workflow with AI
4. Use dashboards/ for plug-and-play templates
```

## 🇻🇳 Tiếng Việt

Tài liệu mã nguồn mở toàn diện giúp thành thạo Excel và Google Sheets — từ công thức cơ bản đến mô hình tài chính nâng cao và tự động hóa bằng AI.

### Nội dung chính

| Phần | Mô tả |
|---|---|
| [`docs/fundamentals/`](docs/fundamentals/) | Công thức cốt lõi, phím tắt, điều hướng giao diện |
| [`docs/advanced/`](docs/advanced/) | Làm việc đa sheet, tối ưu hiệu năng, VBA, dashboard |
| [`docs/domains/`](docs/domains/) | Tài chính, kế toán, phân tích dữ liệu, dự báo, khoa học dữ liệu |
| [`docs/ai-integration/`](docs/ai-integration/) | Tích hợp ChatGPT, Gemini, Claude vào bảng tính |
| [`dashboards/`](dashboards/) | Mẫu dashboard sẵn dùng |
| [`case-studies/`](case-studies/) | Ví dụ thực tế ứng dụng |

### Bắt đầu nhanh

```
1. Xem docs/fundamentals/formulas/ để tra cứu công thức
2. Chọn domain trong docs/domains/ phù hợp nhu cầu
3. Xem docs/ai-integration/ để tăng tốc với AI
4. Dùng dashboards/ như template có sẵn
```
