# Excel Keyboard Shortcuts | Phím tắt Excel

> **Excel** | Fundamentals → Shortcuts

---

## Navigation | Di chuyển

| Shortcut | Action (EN) | Hành động (VI) |
|---|---|---|
| `Ctrl+Home` | Go to cell A1 | Về ô A1 |
| `Ctrl+End` | Go to last used cell | Đến ô cuối dùng |
| `Ctrl+Arrow` | Jump to edge of data | Nhảy đến biên dữ liệu |
| `Ctrl+G` / `F5` | Go To dialog | Hộp thoại Đi tới |
| `Ctrl+F` | Find | Tìm kiếm |
| `Ctrl+H` | Find & Replace | Tìm & Thay thế |
| `Ctrl+Page Up/Down` | Switch sheets | Chuyển sheet |

---

## Selection | Chọn vùng

| Shortcut | Action (EN) | Hành động (VI) |
|---|---|---|
| `Ctrl+Shift+Arrow` | Select to edge of data | Chọn đến biên dữ liệu |
| `Ctrl+A` | Select all | Chọn tất cả |
| `Ctrl+Space` | Select entire column | Chọn cả cột |
| `Shift+Space` | Select entire row | Chọn cả hàng |
| `Ctrl+Shift+End` | Select to last cell | Chọn đến ô cuối |
