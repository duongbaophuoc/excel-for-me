# Google Sheets Keyboard Shortcuts | Phím tắt Google Sheets

> **Google Sheets** | Fundamentals → Shortcuts

---

## Navigation | Di chuyển

| Shortcut | Action (EN) | Hành động (VI) |
|---|---|---|
| `Ctrl+Home` | Go to A1 | Về ô A1 |
| `Ctrl+End` | Go to last cell | Đến ô cuối |
| `Ctrl+Arrow` | Jump to data edge | Nhảy đến biên dữ liệu |
| `Ctrl+F` | Find | Tìm kiếm |
| `Ctrl+H` | Find & Replace | Tìm & Thay thế |

---

## Editing | Chỉnh sửa

| Shortcut | Action (EN) | Hành động (VI) |
|---|---|---|
| `F2` | Edit cell | Sửa ô |
| `Ctrl+D` | Fill down | Sao chép xuống |
| `Ctrl+R` | Fill right | Sao chép sang phải |
| `Ctrl+Z` | Undo | Hoàn tác |
| `Ctrl+Shift+Z` | Redo | Làm lại |
| `Ctrl+Enter` | Fill selection | Điền vùng chọn |
| `Alt+Enter` | New line in cell | Xuống dòng trong ô |
| `Ctrl+;` | Insert date | Chèn ngày |
