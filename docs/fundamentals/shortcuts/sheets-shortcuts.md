# ⌨️ Google Sheets Shortcuts

## 🇺🇸 English

Ctrl + / → Show shortcuts  
Ctrl + Shift + V → Paste values  
Ctrl + Enter → Fill all selected cells  

### Example
Select A1:A5 → type "Test" → Ctrl + Enter

---

## 🇻🇳 Tiếng Việt

Ctrl + / → Xem phím tắt  
Ctrl + Shift + V → Dán giá trị  
Ctrl + Enter → Điền toàn bộ vùng  

### Ví dụ
Chọn A1:A5 → nhập "Test" → Ctrl + Enter
