# Array Formulas / Hàm Mảng Động
- **UNIQUE:** `=UNIQUE(A2:A100)` -> Lọc danh sách khách hàng không trùng lặp.
- **FILTER:** `=FILTER(A2:C100, B2:B100="Hà Nội")` -> Tự động trích xuất toàn bộ hàng có địa chỉ ở Hà Nội.
- **SORT:** `=SORT(FILTER(...), 1, -1)` -> Lọc xong tự động sắp xếp theo thứ tự giảm dần.
