# 🔤 Text Functions

## 🇺🇸 English

### LEFT

#### Syntax
=LEFT(text, num_chars)

#### Example
=LEFT("Excel",2) → "Ex"

---

### TEXTJOIN ⭐

#### Syntax
=TEXTJOIN(", ", TRUE, A1:A3)

#### Example
Join list into string
