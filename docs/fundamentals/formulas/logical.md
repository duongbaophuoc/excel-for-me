# 🧠 Logical Functions

## 🇺🇸 English

### IF

#### Description
Returns value based on condition

#### Syntax
=IF(condition, value_if_true, value_if_false)

#### Example
=IF(A1>10,"High","Low")

---

### IFERROR

#### Syntax
=IFERROR(A1/B1,0)

#### Example
=IFERROR(10/0,0)

---

## 🇻🇳 Tiếng Việt

### IF

#### Mô tả
Trả về giá trị dựa trên điều kiện

#### Cú pháp
=IF(điều_kiện, đúng, sai)

#### Ví dụ
=IF(A1>10,"Cao","Thấp")

---

### IFERROR

#### Ví dụ
=IFERROR(A1/B1,0)
