# Claude for Spreadsheet Analysis
- **Why Claude?** Claude có khả năng đọc các file CSV/Excel cực lớn (lên đến hàng triệu dòng) mà Excel thường bị treo.
- **Use Case:** "Phân tích file báo cáo giao dịch này và tìm ra 5 điểm bất thường (anomalies) trong chi tiêu của tháng trước."
- **Note:** Claude hỗ trợ viết code Python (Artifacts) để vẽ biểu đồ phân tích sâu dữ liệu Excel.
