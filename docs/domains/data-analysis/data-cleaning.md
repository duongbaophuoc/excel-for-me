# Data Cleaning Steps / Quy trình làm sạch dữ liệu
1. **TRIM:** Xóa khoảng trắng thừa.
2. **PROPER:** Viết hoa chữ cái đầu (Chuẩn hóa tên người).
3. **VALUE:** Chuyển đổi văn bản dạng số về số thực để tính toán.
4. **Remove Duplicates:** Data > Remove Duplicates.
