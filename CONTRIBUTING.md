# Contributing Guide | Hướng dẫn đóng góp

## 🇬🇧 English

Thank you for considering contributing to this playbook! Every improvement — from a typo fix to a new case study — is welcome.

### How to Contribute

1. **Fork** the repository
2. **Create a branch**: `git checkout -b feature/your-topic`
3. **Write your content** following the style guide below
4. **Commit**: `git commit -m "Add: [topic] formula / example"`
5. **Push** and open a **Pull Request**

### Content Style Guide

- All documents must be **bilingual (English + Vietnamese)**
- Each formula must include:
  - Syntax
  - Plain-language explanation (EN + VI)
  - At least one practical example with sample data
  - Excel and Google Sheets compatibility note
- Use Markdown tables for sample data
- Keep examples realistic and domain-relevant

### File Naming

- Use **kebab-case**: `financial-functions.md`, `data-cleaning.md`
- Place files in the correct folder matching the directory tree

### Formula Format Template

```markdown
## FORMULA_NAME

**Syntax:** `=FORMULA(arg1, arg2)`

**EN:** What the formula does in plain English.

**VI:** Giải thích công thức bằng tiếng Việt.

### Example | Ví dụ

| A | B |
|---|---|
| data | result |

**Formula:** `=FORMULA(A1)`

**Result:** `value`
```

## 🇻🇳 Tiếng Việt

Cảm ơn bạn đã quan tâm đóng góp cho playbook này! Mọi cải tiến — từ sửa lỗi chính tả đến thêm case study mới — đều được hoan nghênh.

### Cách đóng góp

1. **Fork** repository
2. **Tạo branch**: `git checkout -b feature/chu-de-cua-ban`
3. **Viết nội dung** theo hướng dẫn phong cách bên dưới
4. **Commit**: `git commit -m "Add: [chủ đề] công thức / ví dụ"`
5. **Push** và mở **Pull Request**

### Hướng dẫn phong cách nội dung

- Tất cả tài liệu phải **song ngữ (Anh + Việt)**
- Mỗi công thức phải bao gồm:
  - Cú pháp
  - Giải thích ngôn ngữ tự nhiên (EN + VI)
  - Ít nhất một ví dụ thực tế với dữ liệu mẫu
  - Ghi chú tương thích Excel và Google Sheets
- Dùng bảng Markdown cho dữ liệu mẫu
- Giữ ví dụ thực tế và liên quan đến lĩnh vực

### Quy tắc đặt tên file

- Dùng **kebab-case**: `financial-functions.md`, `data-cleaning.md`
- Đặt file đúng thư mục theo cây thư mục
